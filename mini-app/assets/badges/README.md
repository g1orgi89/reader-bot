# Badge Assets

This directory contains badge images for user gamification.

## Available Badges

- **alice.png** - "Алиса в стране чудес" badge (32x32px for chips, scales to 18px for inline)
- **alice.webp** - WebP optimized version (optional)

## Usage

Badges are rendered in:
- ProfilePage: chip row under avatar (32px) and inline next to username (18px)
- ProfileModal: inline next to username (18px)
- CommunityPage: inline next to usernames in user cards (18px)
